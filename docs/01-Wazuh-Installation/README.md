# Module 1 – Wazuh Installation

## Module Overview

This module documents the deployment of Wazuh as the Security Information and Event Management (SIEM) platform for the SOC Engineering Lab.

The objective was to establish a centralized security monitoring platform capable of collecting, processing, and analyzing security events from Windows endpoints. This deployment serves as the foundation for future modules involving endpoint monitoring, detection engineering, threat hunting, and incident response.

---

## Objectives

- Deploy the Wazuh platform
- Configure the Wazuh Manager
- Connect Windows endpoints using the Wazuh Agent
- Validate secure communication between the agent and manager
- Verify Windows Security Event collection
- Prepare the environment for future SOC engineering activities
